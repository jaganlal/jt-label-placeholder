##Settings
####Usage
```html
	<jt-label-placeholder place-holder={{placeHolderText}}>
	</jt-label-placeholder>
```

```javascript
	$scope.placeHolderText = "Enter date (DD/MM/YY)";
```


####Prerequisite
* Need `npm` & `bower` 
* Run `npm install`
* Run `bower install`


####Required
* `jtLabelPlaceholder` - The directive
* `place-holder` - Place holder for the input element
